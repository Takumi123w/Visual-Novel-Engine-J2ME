Create you own game by edit this package
Unpack and edit story.txt that script for this engine run your VN story

command for story.txt
TITLE: Start menu title

STBG: Start menu background

CH: is character

NAME: was name person who do dialog

TXT: was dialog character

POS: was location character model by use LEFT,RIGHT,FAR_LEFT_FAR_RIGHT AND CENTER

ROT: was rotation character model by use 30,60,90,180,250,360

FLIP: is inverted player rotation use V,H, and VH

CHOICE: is option route example

CHOICE: I want eat that|eat, I don't want eat|not eat
that mean if "I want eat that" then go to "eat" label then if "I don't want eat" then go to "not eat" label

LABEL: is ouput from option choice
example
LABEL: eat
NAME: A person
TXT: Agh... I have been poisoned
GOTO: end

GOTO: is basically jump to certain label without choice

MUS: is music you can input music like .mid file on this
Example
MUS: music.mid

STOPMUS: is stoping music

Basically you can make VN game only with you phone
